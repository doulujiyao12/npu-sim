.. _quickstart:

快速开始
==========
:full-width:

使用NPU-SMI十分简单：

1. 准备 对应 Config 文件，在llm/test/目录下面。
2. 使用命令行启动模型服务。

运行
~~~~~~~~~~~~~
.. code-block:: console

    cd build
    ./npusim --config-file ../llm/test/config_pd_sim.json



.. note::
    在继续这个教程之前，请确保你完成了 :ref:`安装指南 <installation>`, 并确保配置了相关的环境变量。
